# 🧑‍💻 Personal Portfolio - Next.js & Tailwind CSS

Ini adalah landing page portofolio pribadi yang dibangun dengan [Next.js](https://nextjs.org/) dan [Tailwind CSS](https://tailwindcss.com/), dilengkapi dengan berbagai animasi interaktif dan komponen custom.

Website ini bertujuan untuk menampilkan informasi tentang saya, proyek yang telah saya kerjakan, serta menyediakan kontak profesional dalam satu halaman modern dan responsif.

---

## ✨ Fitur Utama

- 🚀 Dibangun dengan Next.js dan Tailwind CSS
- 🎨 Desain responsif untuk mobile, tablet, dan desktop
- 🧩 Komponen animasi custom:
  - `BlurText`, `DecryptedText`, `FadeContent`
  - `ScrollFloat`, `RollingGallery`, `TiltedCard`
  - `PixelCard` berbasis canvas interaktif
- 🌐 Deploy otomatis dengan Vercel
- 🔗 Navigasi yang smooth dan ringan

---

## 🛠️ Teknologi yang Digunakan

- **Next.js** – Framework React modern
- **Tailwind CSS** – Utility-first CSS framework
- **Framer Motion** – Library animasi untuk React
- **Canvas API** – Untuk efek visual seperti `PixelCard`
- **React Hooks & Custom Components**
